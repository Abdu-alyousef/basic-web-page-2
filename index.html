HTML & CSS basic summary

HTML Summary

HTML (HyperText Markup Language) is used to define our page content, structure and meaning.
You don't use it for styling purposes. Use CSS for that instead!

  • HTML uses 'elements' to describe (annotate) content

    • HTML elements typically have an opening content and then closing tag

  • You can also have void (empty) elements like Images

  • You can also configure elements with attributes

    • There's a long list of available elements but you'll gain experience over time, no worries.
    Learn more about all available Html elements on the MDN HTML elements reference.

Css Summary

Css (Cascading Style Sheets) is used for styling your page content.
Style are assigned via property-value paires
you can assign style via the "style" attribute

  • To avoid code duplication, you typically use globel style (e.g via the "style" element) instead

    • allternatively, you can work with external stylesheet files wich you "Link" to

  • When you working with CSS, concepts like "inheritance", "specifacity" and the "Box Model" should be understood.

    • Learn more about all available Css proprties and values on the MDN CSS property reference.
